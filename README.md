## Local Development

Check out the repo and install dependencies in the root folder:

```sh
npm install
npm run build:icons
```

Create environment variable file:

```sh
touch .env
```

local .env variables are stored in 1Password Engineering.

Start application:

```sh
npm run dev
```

## Additional Notes

Start a partykit (websocket) server:

```sh
npx partykit dev
```
